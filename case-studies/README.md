## Case Studies

### [01. Meridian Financial Services](./01-assessment-lending/)
**Phase**: Assessment & Audit (Diagnosis)
**Domain**: Consumer Lending AI ("SmartLend")
**Scale**: 850K+ historical records, 2,000+ daily applications

**Challenge**: Identify sources of bias in an automated loan approval system before implementing interventions.

**Outcome**:
- Comprehensive fairness audit framework
- 34% approval rate disparities identified
- 17 audit dimensions established
- 12 actionable recommendations generated

**Key Deliverables**:
- Audit framework and methodology
- Fairness metrics definitions
- Bias source identification
- Intersectionality analysis
- Implementation checklist

---

### [02. MidCity Community Bank](./02-intervention-lending/)
**Phase**: Intervention & Execution (Treatment)
**Domain**: Consumer Lending
**Scale**: 50,000 historical loan applications (2019-2022)

**Challenge**: Reduce racial and gender disparities in loan approvals while maintaining model performance.

**Results**:
| Metric | Before | After | Change |
|--------|--------|-------|--------|
| Demographic Parity | 0.27 | 0.90 | **+231%** |
| Equal Opportunity | +32% improvement | | |
| Accuracy | 92.3% | 91.6% | **-0.7%** |

**Techniques Applied**:
1. Pre-processing: Disparate Impact Remover
2. In-processing: Adversarial Debiasing
3. Post-processing: Equalized Odds Threshold

**Key Deliverables**:
- Integration workflow
- Implementation guide
- Complete case study with metrics
- Validation framework
- Intersectionality analysis
- Adaptability guidelines
- Organizational guidelines
- Improvement insights

**Key Deliverables**:

- **CEO presentation** (board-ready)

## Progression Logic

Each case study builds on the previous:

| Aspect | Module 1 | Module 2 | Module 3 |
|--------|----------|----------|----------|
| **Core Question** | What bias exists? | How do we fix it? | How do we institutionalize it? |
| **Scope** | Single system audit | Single intervention cycle | Organization-wide transformation |
| **Stakeholders** | Data scientists, auditors | ML engineers, product teams | Executive leadership, governance |
| **Output** | Diagnostic report | Measured intervention results | Strategic governance model |
| **Maturity Level** | Aware | Capable | Governed |
